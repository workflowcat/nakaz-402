# drafts/ — робочі проєкти правок до Положення

Тут зберігаються машино-читабельні чернетки амендментів до Наказу
МОУ № 402. **Це не офіційні нормативно-правові акти, юридичної сили вони
не мають.** Сюди потрапляють ескізи, які громада, юристи МОУ,
ЦВЛК-експерти та адвокаційні організації готують і обговорюють — до
того, як хтось із них стане справжнім наказом.

Авторитетна редакція Наказу № 402 — на
[zakon.rada.gov.ua](https://zakon.rada.gov.ua/laws/show/z1109-08).

## Що описує один файл

Один файл `*.md` = один проєкт правки. У YAML-заголовку — список
операцій над конкретними пунктами:

| Тип операції | Що робить | Що треба вказати |
|---|---|---|
| `redaction` | Викласти пункт у новій редакції | `target` (ID пункту), `new_text` |
| `insert`    | Додати новий пункт | `after` або `before` (сусід), `new_id` (новий номер), `text` |
| `repeal`    | Виключити пункт | `target` |
| `restore`   | Поновити раніше виключений пункт | `target`, `new_text` |

Кожен пункт має стабільний ID на кшталт `polozhennia.r1.gl1.p1.4` —
`розділ 1, глава 1, пункт 1.4`. Не змінюється, навіть якщо пункт
переноситься або редагується.

## Як подати свій проєкт

1. Створіть гілку: `draft-<рік>-<короткий-опис>` (наприклад
   `draft-2026-trembita-logging`).
2. Додайте файл `drafts/<рік>-<NNN>-<slug>.md`. Орієнтуйтеся на наявні
   приклади — формат однаковий.
3. Залишіть `status: draft` (потім можна підвищити до `review`, а далі —
   `adopted` якщо МОУ фактично прийняв подібний наказ, або `rejected`).
4. Відкрийте Pull Request. Сайт автоматично проведе перевірки:
   - чи існують пункти, на які ви посилаєтеся;
   - чи унікальний `new_id` для нового пункту;
   - чи не конкурує ваш проєкт з іншим відкритим;
   - чи не вжито застарілий термін (наприклад «аеромобільні» замість
     «Десантно-штурмові»).
5. Як PR пройде ревʼю — він з'явиться на `/drafts/<slug>/` з наочною
   різницею «до / після», списком зауважень і автоматично згенерованим
   формальним текстом «Внести зміни...».

## Статуси

| Статус | Що означає |
|---|---|
| `draft` | Чернетка. Автор працює над текстом, обговорення ще не почалося |
| `review` | На огляді стейкхолдерами. Текст стабілізувався, чекає на коментарі |
| `adopted` | МОУ фактично прийняв подібну редакцію. У полі `adopted_order` — № реального наказу |
| `rejected` | Знято з розгляду (не пройшов узгодження, або застаріло) |
