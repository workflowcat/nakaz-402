# Додаток 2. Пояснення щодо застосування статей Розкладу хвороб

> 🟡 **Stub.**

Цей додаток містить тлумачення термінів і критерії, за якими лікарі ВЛК
застосовують конкретні статті Розкладу хвороб. Структурно — це коментар
до кожної статті або групи статей.

## Рекомендована структура файлу при наповненні

- Розбити по статтях (`#stattia-1`, `#stattia-2`, ...).
- Для кожної статті — критерії, особливості огляду, посилання на МКХ-10,
  додаткові обстеження.
- При зміні Розкладу хвороб — паралельно оновлювати Пояснення (в одному PR).
