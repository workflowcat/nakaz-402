# Додаток 3. Таблиці додаткових вимог до стану здоров'я (ТДВ)

> 🟡 **Stub.** Це другий великий «конфіг» Положення поряд з Розкладом хвороб.

ТДВ — це додаткові медичні вимоги для **окремих категорій служби**: льотний
склад, морський склад, водолази, парашутисти, спецпризначенці, оператори
БпЛА тощо. Якщо Розклад хвороб відповідає на питання «чи придатний взагалі»,
то ТДВ — «чи придатний саме до цього виду діяльності».

## Рекомендована структура

Кожна таблиця ТДВ має ту саму ідею, що й Розклад хвороб — це матриця:
**вид діяльності × медична вимога × норма / неприпустиме значення**.
Тобто природно ляже в YAML (один файл — одна таблиця ТДВ).

Можлива структура:

```
03-tdv/
├── README.md
├── meta.yaml                ← перелік таблиць і граф
├── tdv-01-lotnyi-sklad.yaml
├── tdv-02-vodolazy.yaml
├── tdv-03-parashutysty.yaml
├── tdv-04-spetspryznachennia.yaml
└── ...
```

При наповненні — за тією самою логікою, що й Розклад хвороб: одна вимога =
один рядок YAML, поправка = один PR.
