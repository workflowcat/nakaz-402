## Розділ II. Медичний огляд

23 глави, які регламентують процедури медогляду для всіх категорій осіб у системі ЗСУ.
**Всі мають структурні стаби з ID, можна одразу посилатись.** Зміст заповнюється
поступово через PR-флоу. Повний перелік і навігація — [у README Розділу II](02-medychnyi-oglyad/).

## Розділ III

**Виключено** одним з амендментів. Зберігається як історичний запис у git-історії
та `meta/amendments.yaml`.

## Що значить «стаб»

Стаб — це файл з повною frontmatter-метаданими (id, parent, status, source) і
посиланням на офіційний текст, але без розгорнутого тексту пунктів. Структурно
він повноцінний — на нього вже можна посилатись з amendments.yaml або diff inspector'а,
і він з'являється у nav. Заповнення — через PR за зразком уже готових Глав 1 і 2.

Авторитетним джерелом залишається [текст на zakon.rada.gov.ua](https://zakon.rada.gov.ua/laws/show/z1109-08).
